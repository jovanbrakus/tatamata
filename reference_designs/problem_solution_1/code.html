Solve for x in the interval [0, 2π]:

2sin²(x) - 3cos(x) = 0

lightbulb Hint

Try converting everything to cosine.

timer Avg. Time

~ 8 minutes

checklist Step-by-Step Solution

3 Steps
1

Apply Pythagorean Identity

Since the equation contains both sin and cos, we use the identity sin²(x) = 1 - cos²(x) to express the equation solely in terms of cosine.
2(1 - cos²(x)) - 3cos(x) = 0
2

Expand and Rearrange

Distribute the 2 and rearrange the terms to form a standard quadratic equation format.
2 - 2cos²(x) - 3cos(x) = 0
-2cos²(x) - 3cos(x) + 2 = 0
Multiply by -1 to simplify:
2cos²(x) + 3cos(x) - 2 = 0
warning

Common Mistake

Be careful with signs when multiplying by -1. Ensure all terms flip signs correctly.

3

Solve the Quadratic for cos(x)

Let u = cos(x). We now have 2u² + 3u - 2 = 0. Using the quadratic formula or factoring:
(2u - 1)(u + 2) = 0
Solution 1
cos(x) = 1/2
Solution 2
cos(x) = -2
Invalid (Range [-1, 1])
check_circle Final Answer
x = π/3, 5π/3
